# Developer Tools - Future Features

This document tracks planned features for the REPLAYER Developer Tools menu.

---

## Current (Implemented)

- [ ] Start/Stop Raw Capture
- [ ] Analyze Last Capture
- [ ] Open Captures Folder

---

## Planned - WebSocket Tools

- [ ] Connection Status Panel - Shows detailed connection info (socket ID, state, latency)
- [ ] Live Event Log Viewer - Real-time scrolling view of events as they arrive
- [ ] Force Reconnect - Manual reconnection trigger
- [ ] Event Filter - Show/hide specific event types in live view

---

## Planned - State Tools

- [ ] Dump GameState - Save current local state to JSON file
- [ ] View Server State - Show last playerUpdate data in panel
- [ ] Compare Local vs Server - Side-by-side drift analysis
- [ ] State History - Timeline of state changes

---

## Planned - Logging Tools

- [ ] Open Log Folder
- [ ] Set Log Level (DEBUG/INFO/WARN/ERROR submenu)
- [ ] Export Session Logs - Bundle all logs from current session
- [ ] Log Search - Find specific entries

---

## Planned - Analysis Tools

- [ ] Event Statistics - Charts of event frequency over time
- [ ] Latency Histogram - Distribution of tick intervals
- [ ] Protocol Documentation Generator - Auto-generate docs from captures
